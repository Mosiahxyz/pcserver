Placa-Mãe

A placa mãe é a parte do computador onde todos os componentes são interligados. Isso é, é a ponte de conexão entre toda a operação do seu sistema. Ela é a peça principal do seu computador e contém diversos soquetes que a conectam a outras placas, como a placa de som ou de vídeo, por exemplo. A placa-mãe escolhida possui: Suporte para processador Intel core i7 / processador Intel Core i5/ processador Intel Core i3/ processador Intel Pentium/ processador Intel Celeron no pacote LGA1151. O cache L3 varia com CPU.
